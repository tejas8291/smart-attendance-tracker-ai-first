# Smart Attendance Tracker

An AI-first engineering project demonstrating attendance tracking with Flutter and FastAPI.

## Problem

Many attendance systems only support check-in and check-out.

Common issues:

- Location tracking is not reliable
- Users may go offline
- Data can be lost
- Companies cannot verify field activity

## Solution

Build a smart attendance system that:

- Supports Login
- Check In
- Check Out
- Location Tracking every 15 minutes
- Offline Storage
- Automatic Sync when internet becomes available

## Tech Stack

### Frontend
- Flutter
- Riverpod
- Dio

### Backend
- FastAPI
- Swagger/OpenAPI

### Storage
- Hive / Isar
